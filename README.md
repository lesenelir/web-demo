### 项目主页更新步骤：

1. 切换到gh-pages分支操作
2. gh-pages分支下：
   - `git add .`
   - `git commit -m "x"`
   - `git push -u origin gh-pages`
3. 合并gh-pages分支到master分支：
   - 切换到主分支 `git checkout master`
   - 在master分支下合并 `git merge gh-pages`
4. 更新主分支：
   - `git push -u origin master`
